Pricing

I provide three kinds of manga translation:

  • Text Scripts: Starting at $1.5 per page.
    • The delivery format will be either as a script or as an annotated PDF.
    • Please note that the price may increase if the work contains an exceptionally large amount of text.
    • Please tell me if you'd like SFX, prefaces, or afterwords translated as well.1,2
  • Lettering Only: $1–$2 per page.
    • The delivery format will be as a zipped file containing the translated pages as PNGs (+ the Adobe Photoshop and InDesign source files, if requested).
    • Please note that works requiring a lot of editing may cost more.
    • SFX are not included.
  • Full Translation: Starting at $2.5 per page.
    • The delivery format will be as a zipped file containing the translated pages as PNGs.
    • The price will increase corresponding to the amount of text and to the amount of editing required.
    • SFX are not included.

The price for novel translations is as follows:

  • Text: $0.03–$0.07 per character.
  • Scans: $4–$8 per scanned page.
The delivery format will be as a PDF professionally typeset using the LaTeX novel package.

How long does it usually take?

Manga translations containing 20–50 pages take about a week. The usual timeframe for novel translations is 1–2 pages per day.
